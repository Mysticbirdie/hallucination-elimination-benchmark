## Reporting a Vulnerability

If you discover a security vulnerability in this benchmark, please report it privately before disclosing it publicly.

### How to Report

Send an email to: security [at] airtrek [dot] ai

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Security Best Practices

- Keep API keys secure and out of version control
- Use environment variables for sensitive configuration
- Review model outputs for unintended content
- Validate all external inputs
